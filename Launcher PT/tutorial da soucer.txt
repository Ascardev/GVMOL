
parte 1 conf no servidor

Abra o Launcher.sln (caso esteja em Debug, mude para Release) e procure por essa linha


string Servidor = "http://127.0.0.1/Launcher/" ;  
nessa linha você vai alterar 127.0.0.1 para o IP do seu Host (normalmente é o mesmo IP que você entra no Host pela Área Remota)

Em alguns casos (como no meu) o Host tem algumas frescuras (usar IP diferente do qual você usa pra se conectar na maquina, ter limite de portas, etc cada um funciona de um jeito)
no meu caso eu tenho de adicionar a porta junto ao IP caso contrario o Launcher não "funciona" ficando assim: (porta 9999 como exemplo)


string Servidor = "http://127.0.0.1:9999/Launcher/" ;  
feito isso só Compilar e Pronto.
(oque, é só isso?)
sim, está pronto seu Launcher.
